# 销售场景战卡学习系统

面向销售、售前、市场人员的碎片时间学习工具。

## 功能

- **六大场景战卡**：每张战卡含 8 个模块（场景一句话、目标客户、客户痛点、推荐方案、话术、必问问题、试点落地、禁止承诺）
- **闪卡练习**：翻转卡片记忆模式，支持「已掌握/再练一次」自适应学习
- **知识测验**：单场景或全场景随机出题，结果可复盘
- **话术速查**：快速查看各场景首次拜访话术
- **学习进度**：本地持久化记录每个知识点的掌握情况

## 使用方式

直接用浏览器打开 `index.html`，无需安装任何依赖。

## 数据更新

所有场景内容在 `js/data.js` 的 `SCENARIOS` 数组中，按场景替换为真实 PPT 内容即可。
